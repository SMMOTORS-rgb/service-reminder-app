# Hosting on GitHub Pages

Follow these steps to host your **Sivamalai Motors Management Suite** on the internet for free using GitHub Pages.

## Step 1: Create a GitHub Repository
1. Log in to [GitHub](https://github.com/).
2. Click the **+** icon in the top right and select **New repository**.
3. Name it (e.g., `sivamalai-motors`).
4. Set it to **Public**.
5. Click **Create repository**.

## Step 2: Upload Your Files
1. On your new repository page, click **uploading an existing file**.
2. Drag and drop all your files from the `service-reminder-app` folder:
   - `index.html` (The main menu)
   - `service.html`
   - `sales.html`
   - `attendance.html`
3. Scroll down, add a comment like "Initial upload", and click **Commit changes**.

## Step 3: Enable GitHub Pages
1. Go to the **Settings** tab in your repository.
2. In the left sidebar, click **Pages**.
3. Under **Branch**, select `main` (or `master`) and click **Save**.
4. Wait about 1-2 minutes. A link will appear at the top saying: 
   > "Your site is live at `https://yourusername.github.io/sivamalai-motors/`"

## Step 4: Verify Your App
1. Click the link provided by GitHub.
2. Enter your PIN **`8020`**.
3. Your app is now live and will sync with your Google Sheets just as it did locally!

---

> [!IMPORTANT]
> **Privacy Note**: Since your repository is Public, anyone who knows the URL can see your code. However, they still need the PIN `8020` to access the data through the interface.
